Flexbox is a layout model in CSS that makes it easier to create responsive and flexible layouts. It allows elements within a container to be automatically arranged depending on the available space, and it makes it easy to align and distribute space among elements in a container.

display: flex;

This sets the element to which it is applied as a flex container. In other words, it creates a flexbox. This means that the element's child elements will be automatically arranged and resized within the container according to the flexbox layout model.

justify-content: space-between;

The justify-content property is used to align flex items within a flex container along the main axis. When it is equal to space-between, it distributes the blank space evenly between (not around) elements. Alternatives include center, space-around, and space-evenly.

align-items: center;

The align-items property can be used to control the alignment of flex items within a flex container along the cross (i.e., not main) axis. When it is equal to center, by default, it centers items vertically in the container. Alternatives include start, end, and stretch.

flex-wrap: wrap;

This is used to control the behavior of flex items when there is not enough space to fit them all within a flex container. It causes the flex items to wrap onto multiple lines if necessary, rather than shrinking them. Alternatives include nowrap and wrap-reverse.

other CSS properties

  • flex-direction (default: row)
  • order (default: 0)
  • align-self (default: auto)
  • flex-grow (default: 0)
  • flex-shrink (default: 1)
